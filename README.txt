Project Description
===================

This project is focused on development of Client-side PowerShell scripts for backup and restore of the SharePoint Term Store.

1. The TaxonomyBackup script backs up complete Term Store to XML file including the [System] Term Group.
2. The TaxonomyRestore script is restores the Term Store from the XML file created by the TaxonomyBackup script. The [System], [Search Dictionaries], and [People] Term Groups are not restored.


This script is based on a [SharePoint 2010 Server side version][1].
[1]: http://taxonomyscripts.codeplex.com
